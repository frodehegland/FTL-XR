# Asset to URL

* format: https://frodehegland.github.io/FTL-XR/?`yourfile.glb`

> Example: [https://frodehegland.github.io/FTL-XR/?test-frode-table.glb](https://frodehegland.github.io/FTL-XR/?test-frode-table.glb)
